# AI Agent 应用分享

这是一个用于“AI Agent 应用分享”的 Vite + React + TypeScript 单页展示项目。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物位于 `dist/`，该目录不纳入 Git 版本管理。

## 内容

页面包括首页、分享路线、ChatGPT 与 Codex 的区别、Codex 环境准备、使用技巧、AGENTS.md、Skills、MCP、CLI、GitHub 与 Netlify、自动化任务、人类引导、Fork / Worktree 和总结。
